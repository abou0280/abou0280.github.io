Html Tags

br tag
The break tag is used to separate lines of text from long
overstretched lines across a website by breaking lines of text and
continuing on the next line to create a more coherent paragraph.
abbr tag
The abbreviation tag is used to shorten any long
abbreviation or acronym into a short form.
b tag
The bold tag is used to bolden any text without marking it as important.
The bold tag can therefore be used for common text that you want to make more apparent.
article
The article tag helps specify individual
texts to be in their own section. Article tag is meant to be used for texts
that contain their own special information inside.
blockquote
The blockquote tag is used to refer to another website. It can be used to quote
certain information from the website while adding the link to the website externally.
